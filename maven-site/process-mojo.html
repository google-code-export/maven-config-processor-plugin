config-processor:process

Full name:

com.google.code.maven-config-processor-plugin:maven-config-processor-plugin:1.3-SNAPSHOT:process

Description:

Generates modified configuration files according to configuration. Includes, excludes and modify properties.

Attributes:

  • Requires a Maven 2.0 project to be executed.
  • Binds by default to the lifecycle phase: process-resources.

Required Parameters

Name Type Since Description
outputDirectory File 1.0 Output directory of the generated files.
Default value is: ${project.build.directory}.
transformations List 1.0 File transformations to be performed.
useOutputDirectory boolean 1.0 Indicate if should prefix file paths with the outputDirectory configuration property.
Default value is: true.

Optional Parameters

Name Type Since Description
encoding String 1.1 Encoding to use when reading or writing files.
Default value is: ${project.build.sourceEncoding}.
indentSize Integer 1.2 Indentation size as the number of whitespaces to use when formatting.
Default value is: 4.
lineWidth Integer 1.2 Maximum line width of the generated files to use when formatting.
Default value is: 80.
namespaceContexts Map 1.2 Namespace contexts for XPath expressions. Mapping in the form prefix => url
specificProperties File 1.0 File to load aditional specific properties for plugin execution.
